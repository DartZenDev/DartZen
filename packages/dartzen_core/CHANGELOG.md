# Changelog

## 0.0.1

- Initial release of `dartzen_base`.
- Added `ZenResult`, `ZenSuccess`, `ZenFailure`.
- Added `ZenError` hierarchy (`ZenValidationError`, `ZenNotFoundError`, etc.).
- Added `BaseResponse` contract.
- Added Core Value Objects (`ZenTimestamp`, `ZenLocale`, `EmailAddress`, `UserId`).
- Added `ZenGuard` and `ZenTry` utilities.
