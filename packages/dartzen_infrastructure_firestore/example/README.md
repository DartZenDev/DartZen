# Firestore Infrastructure Example

This directory contains example code demonstrating how to use `dartzen_infrastructure_firestore` in your application.

## Running the Example

This example is for documentation purposes only. To run it in a real application:

1. Initialize Firebase in your app
2. Connect to a Firestore instance
3. Wire the repository to your domain services

## Key Concepts Demonstrated

- Creating a `FirestoreIdentityRepository`
- Saving and loading identities
- Using lifecycle transitions
- Running cleanup operations
- Error handling with `ZenResult`
- Using the adapter as an `IdentityProvider`

See [main.dart](main.dart) for the complete example.
